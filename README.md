# Prepstr

This repository contains a Streamlit-based mock test platform (Prepstr) implementing the core features described in the README.

How to run locally:

1. python -m venv venv
2. source venv/bin/activate
3. pip install -r requirements.txt
4. streamlit run app.py

Notes:
- The app uses a local SQLite DB at data/prepstr.db which is initialized on first run.
- Default admin user is created on first run: username=PradipT0928
